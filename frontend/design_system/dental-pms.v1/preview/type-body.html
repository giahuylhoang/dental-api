--text-lg / --leading-relaxed

Patient notes and clinical observations use this size for comfortable reading during long shifts. The relaxed line height reduces eye strain.

--text-base / --leading-normal (default body)

Standard body copy for forms, descriptions, and secondary content. This is the default reading size across the PMS interface.

--text-sm / --leading-normal

Secondary labels, helper text, and metadata. Used for timestamps, record IDs, and supporting information.

--text-xs / --leading-normal

Micro labels, badge text, and table column headers. Use sparingly.
